# Code Playground

pnpm monorepo 学习项目，包含多个子项目。

## 项目结构

- `react-ts-demo/` — React + TypeScript + Vite（练习 React 组件模式）
- `ts-node-demo/` — TypeScript + ts-node（练习 TS 语法）
- `docs/` — 学习笔记（按归属分类）
  - `js-ts-fundamentals.md` — JS/TS 语言基础，浏览器 API 等
  - `react-hooks.md` — React 基础（Props、State、useState、useReducer、useContext、useEffect、useMemo、useCallback、useRef）
  - `react-hooks-advanced.md` — React 进阶（执行阶段、Commit 详解、flushSync、命令式vs声明式、React 19 新 API、zustand、自定义 Hook）
  - `typescript-ecosystem-notes.md` — 工具链/生态（pnpm、Vite、tsconfig）
  - `zod-and-ts-types.md` — Zod Schema 验证与 TS Utility Types（Partial、Pick、Omit、Required）
  - `react-router.md` — React Router v7（路由方式、Data Router、历史记录栈对比、选型建议）
  - `frontend-tech-stack.md` — 前端技术栈全景（Vite、React、Zustand、TanStack Query、Tailwind CSS、shadcn/ui、React Hook Form + Zod、dnd-kit）
  - `dnd-kit.md` — dnd-kit 拖拽交互（DndContext、useDraggable、useDroppable、useSortable、onDragEnd 数据流、kanban 实践）
  - `tailwind-shadcn.md` — Tailwind CSS v4 + shadcn/ui（安装、cn()工具、CVA模式、暗色模式、最佳实践）
  - `zustand.md` — Zustand 状态管理（基本用法、选择器、中间件、异步操作、与 kanban 项目实例）
  - `vite.md` — Vite 构建工具（双引擎架构、vite.config.ts 常用配置、环境变量、分包策略）
  - `interview-frontend.md` — 前端面试题（Event Loop、闭包与 Hook、原型链、Fiber、虚拟 DOM、响应式原理、Webpack vs Vite、构建速度优化）
