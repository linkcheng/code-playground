# Code Playground

前端学习项目，pnpm monorepo 管理。

## 子项目

| 目录 | 说明 | 技术栈 |
|------|------|--------|
| `kanban/` | 看板应用（拖拽、任务管理） | React + TypeScript + Vite + Tailwind + shadcn/ui |
| `react-ts-demo/` | React 组件模式练习 | React + TypeScript + Vite |
| `ts-node-demo/` | TypeScript 语法练习 | ts-node |

## 学习笔记

`docs/` 目录下按主题分类：

- **语言基础** — JS/TS 语言特性、CSS 基础
- **React** — Hooks 基础与进阶、Router、面试题
- **工具链** — Vite、pnpm、tsconfig、TypeScript 生态
- **UI 库** — Tailwind CSS、shadcn/ui、dnd-kit
- **状态管理** — Zustand、Zod 校验
